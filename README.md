# Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### Env
Remember to set the variable `SPOTIFY_TOKEN_API` in the file `.env.local`

- [Access Token](https://developer.spotify.com/documentation/web-api/concepts/access-token)
